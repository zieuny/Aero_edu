# AeroAuto Pilot Design System: Hanwha Aerospace Edition

## Visual Identity
This design system is inspired by the **Hanwha Aerospace** corporate identity, emphasizing precision, reliability, and modern aerospace technology.

## Color Palette
- **Hanwha Orange (#F37321)**: The primary brand color, used for calls to action, active navigation states, and highlighting key data points.
- **Hanwha Black (#000000)**: Used for core structural elements, primary text, and deep contrast sections.
- **Hanwha Dark Gray (#1A1A1A)**: Secondary background color for sidebar and header accents.
- **Precision Gray (#707070)**: Used for secondary text, labels, and icons.
- **Industrial White (#F8F8F8)**: The main canvas background color to ensure a clean, airy workspace.

## Typography
- **Primary Font**: **Inter** (Sans-serif)
  - Headings: Bold/ExtraBold (700-900) with tight tracking for a "solid" technical feel.
  - Body: Regular/Medium (400-500) for high legibility.
  - Data: **JetBrains Mono** for technical logs and code snippets.

## Layout & Components
- **Grid Structure**: Rigid, clear grid layouts to reflect aerospace engineering discipline.
- **Borders & Corners**: Subtle rounding (8px - 12px) instead of large "bubbly" curves to maintain a professional, industrial look.
- **Transitions**: Professional, swift motion effects (fade-in, slight slide) to reflect system responsiveness.
- **Icons**: Clean, geometric icons (Lucide) styled with corporate colors.
